# プラットフォーム
以下は SIMNOS がサポートするプラットフォームの一覧です。ここに記載されているすべてのプラットフォームは Netmiko を使用してログインのテストが行われていますので、少なくとも正しくログインできることが保証されています。各プラットフォームのステータスは以下の記号で示されます:

- ✅: 完全サポート
- ❌: 現在動作していません

!!! note
    プラットフォームに問題が見つかった場合は、[GitHub リポジトリ](https://github.com/Route-Reflector/simnos/issues)で Issue を作成してください。
