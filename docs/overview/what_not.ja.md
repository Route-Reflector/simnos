# SIMNOS でないもの

SIMNOS はシミュレータであり、ネットワークのコントロールプレーン、データプレーン、マネジメントプレーンのいずれもエミュレートしません。単にコマンドを入力として受け取り、事前定義された出力で応答するだけです。動的な出力を追加する計画はありますが、基盤となるプロトコルの動作をエミュレートする予定はありません。

BGP や LLDP などの基盤プロトコルをエミュレートするネットワークエミュレータをお探しの場合は、以下が参考になるかもしれません:

-  [GNS3](https://www.gns3.com/)
-  [EVE-NG](https://www.eve-ng.net/)
-  [Cisco Packet Tracer](https://www.netacad.com/es/courses/packet-tracer)

このライブラリの考え方は、ネットワーク機器と対話する自動化ソフトウェアに組み込み、高レベルで動作をエミュレートできるようにすることです。
