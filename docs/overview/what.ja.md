# SIMNOS の目標
SIMNOS の主な目標は、ネットワークデバイスの SSH 接続と動作を軽量かつ簡単にシミュレートすることです。以下は SIMNOS の目標の一部です:

-  数千台のサーバーを起動してアプリケーションの負荷テストを実施する。
-  ネットワークオペレーティングシステムのコマンドラインインターフェース（CLI）操作をシミュレートする。
-  カスタム NOS プラグインを作成するための高レベル API を提供する。
-  Docker コンテナで実行してインフラストラクチャとの統合を簡素化する。
-  SIMNOS CLI ツールを活用して素早くシミュレーションを実行・プロトタイプする。
