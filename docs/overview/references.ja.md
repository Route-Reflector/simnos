# SIMNOS が参考にした・取り入れたプロジェクト

- [sshim](https://pythonhosted.org/sshim/) - SSH 自動化クライアントのテストとデバッグのためのライブラリ
- [PythonSSHServerTutorial](https://github.com/ramonmeza/PythonSSHServerTutorial) - paramiko ベースの SSH サーバー作成チュートリアル
- [fake-switches](https://github.com/internap/fake-switches) - プラグイン方式のスイッチ/ルーターコマンドラインシミュレータ
- [ncs-netsim](https://developer.cisco.com/docs/nso/guides/#!the-network-simulator) - デバイスネットワークをシミュレートするツール
- [cisshgo](https://github.com/tbotnz/cisshgo) - テスト目的でネットワーク機器をエミュレートする並行 SSH サーバー
- [scrapli-replay](https://pypi.org/project/scrapli-replay/) - SSH プログラムの簡単なテストとセミインタラクティブな SSH サーバー作成のためのツール
